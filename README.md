\# 📄 Chat with Your PDF (RAG App)



A simple AI-powered application that allows users to upload a PDF and ask questions about its content using Retrieval-Augmented Generation (RAG).



Built with \*\*Streamlit, LangChain, HuggingFace, and FAISS\*\*.



\---



\## 🚀 Features



\- 📂 Upload any PDF file

\- 🔍 Extract and split document text

\- 🧠 Generate embeddings using HuggingFace

\- ⚡ Store and retrieve using FAISS vector database

\- 💬 Ask questions and get context-based answers

\- 🎯 Uses RAG (Retrieval-Augmented Generation)



\---



\## 🛠️ Tech Stack



\- \*\*Frontend\*\*: Streamlit  

\- \*\*LLM\*\*: HuggingFace (`flan-t5-large`)  

\- \*\*Embeddings\*\*: sentence-transformers  

\- \*\*Vector DB\*\*: FAISS  

\- \*\*Framework\*\*: LangChain
